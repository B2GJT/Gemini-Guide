## 주요 내용

### 1. Gemini Flash / Pro 모드 설정하기
프롬프트 입력란 오른편 [모드] 버튼을 클릭해 Flash 또는 Pro 모드를 선택할 수 있습니다.

### 2. Google Workspace 앱 연동하여 답변 받기
프롬프트 입력창에 `@`를 입력하면 Gmail, Google Drive 등 Workspace 앱과 연동하여 답변을 받을 수 있습니다.

### 3. 이미지 생성, Canvas 기능 사용하기
프롬프트 입력란 [도구] 버튼을 클릭해 이미지 생성 또는 Canvas 기능을 사용할 수 있습니다.

### 4. 설정 - 시스템 설정
왼쪽 메뉴 → 설정 및 도움말에서 테마, 언어 등 시스템 환경을 설정할 수 있습니다.

### 5. Gemini에게 저장하도록 요청한 정보 메뉴 확인
왼쪽 메뉴 → 설정 및 도움말 → Gemini 요청 사항에서 기억시킨 정보를 확인하고 관리할 수 있습니다.

### 6. Google Workspace 앱 연동 확인 및 켜기
Workspace 앱 연동 상태를 확인하고 필요한 앱을 활성화하는 방법을 안내합니다.

### 7. 대답 재작성 및 Docs 내보내기 & Gmail 초안 작성
Gemini의 답변을 재작성하거나 Google Docs로 내보내고, Gmail 초안을 자동 작성하는 방법을 안내합니다.
