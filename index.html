설정법 가이드는 Gemini Pro (= Google AI Studio Pro) 3.0 기준으로 작성되었습니다.

1 Gemini flash / pro 모드 설정하기

1
프롬프트 입력란 오른편의 [모드]를 클릭하여 원하는 모드를 선택/변경합니다.
프롬프트 입력란 오른편 [모드] 버튼 클릭 → 원하는 모드 선택
⚡ 빠른 모드 (flash)
빠른 속도로 답변
🧠 사고 모드
Thinking 단계를 거쳐 복잡한 문제를 논리적으로 답변
🚀 pro 모드
고난도 코딩·수학 등 전문 작업에 최적화된 최고 성능

2 Google Workspace 앱 연동하여 답변 받기

Google Workspace 앱 연동을 켠 상태에서 사용해주세요! (설정 방법은 STEP 6 참고)
1
프롬프트 입력창에 @ 를 입력하면 '기타 앱' 목록이 표시됩니다.
2
목록에서 필요한 Google Workspace 앱을 선택하여 연동합니다.
프롬프트 입력창에 @ 입력 → 기타 앱 목록에서 Google Workspace 앱 선택

3 이미지 생성, Canvas 기능 사용하기

1
프롬프트 입력란의 [도구]를 클릭하여 이미지 생성 또는 Canvas 기능을 선택합니다.
프롬프트 입력란 [도구] 클릭 → 이미지 생성 또는 Canvas 기능 선택

4 설정 - 시스템 설정

1
Gemini 앱 화면 왼편에서 아래 경로로 이동합니다.
설정 및 도움말 → 테마 → 시스템
왼쪽 메뉴 → 설정 및 도움말 → 테마 → 시스템 선택

5 Gemini에게 저장하도록 요청한 정보 메뉴 확인

1
Gemini 앱 화면 왼편에서 아래 경로로 이동합니다.
2
클릭 후 Gemini에게 기억시킬 필요한 정보를 입력합니다.
설정 및 도움말 → Gemini 요청 사항
왼쪽 메뉴 → 설정 및 도움말 → Gemini 요청 사항 → 기억시킬 정보 입력

6 Google Workspace 앱 연동 확인 및 켜기

1
Gemini 앱 화면 왼편에서 아래 경로로 이동합니다.
2
Google Workspace 앱 연동 토글을 켜기(ON) 상태로 변경합니다.
설정 및 도움말 → 연결된 앱 → Google Workspace
설정 및 도움말 → 연결된 앱 → Google Workspace → 토글 ON으로 변경

7 대답 재작성(재확인) 및 Docs 내보내기 & Gmail 초안 작성

1
작성된 대답 아래의 점 세 개 아이콘 (더보기)을 클릭합니다.
2
대답 재확인, Docs 내보내기, Gmail 초안 작성 등 필요한 메뉴를 선택하여 작업을 진행합니다.
대답 하단 점 세 개 아이콘 클릭 → 재확인 / Docs 내보내기 / Gmail 초안 작성 선택
